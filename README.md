# Endiks
This is an Ecommerce Marketplace app using React.js, Next.js, Redux, Node.js, Express, Mongoose and some other packages.
### Usage
```bash
$ npm install
```
```bash
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:3000
```
### Config Vars
Open "config/default.json" and add the relavant config vars
